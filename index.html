What aMUSEme is all about

aMUSEme is an inspirational site that is meant to help creative people express themselves and find music, articles, and other artistic outlets to better the soul.

Use the music search to get the good vibes flowing while you find some inspirational material through the compilation search feature.

Feel free to bookmark this page for future and continuous use :).

This project is brought to you by The Funtastic 4 Team. Check out our GitHub page if you have any comments, suggestions, or issues.
